# LeetMentor

LeetMentor is a lightweight browser extension designed to assist users while solving LeetCode problems. It detects the active problem and provides problem explanations, step-by-step breakdowns, and multi-language solution code directly from a popup window.

## Features

- **Automatic Problem Detection:** It instantly detects the active LeetCode problem title and the programming language currently selected in the editor.
- **Problem Explanation:** Generates clean, beginner-friendly explanations tailored to the detected problem.
- **Step-by-Step Breakdown:** Provides a structured, logical walkthrough to guide you from understanding the problem to forming the solution.
- **Multi-Language Code Generation:** Automatically produces optimized solution code in the same language you're using on LeetCode, or any language you select from the popup.
- **Complexity Analysis:** Each solution includes clear time and space complexity analysis, helping you understand performance and trade-offs.
- **Clean UI & Syntax Highlighting:** Features a dark-themed, smooth reading experience, and syntax highlighting powered by highlight.js.
- **Lightweight & Fast:** Built with pure HTML/CSS/JS—no heavy frameworks—ensuring instant responses and minimal resource usage.

## Technologies Used

- HTML, CSS, JavaScript
- highlight.js (syntax highlighting)
- Gemini API integration (for explanation, code generation and complexity analysis)
